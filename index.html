ESP32 · ADS-B · Flight Radar

Sugarfoot'sFlight Radar

Flash your ESP32 and watch the skies — no software required.

⚠ Web Serial requires Google Chrome or Microsoft Edge. Please open this page in Chrome to install.

What you need

  • ESP32 DevKit V1 board
  • USB cable (data — not charge-only)
  • Google Chrome or Microsoft Edge browser
  • TFT display connected to your ESP32
  • Free account at opensky-network.org (optional, for more data)

Flash Firmware

01

Plug in your ESP32

Connect it to your computer with a USB data cable. If Windows asks to install drivers, let it.

02

Click Install below

Chrome will ask you to select a serial port — pick the one labelled CP2102 or CH340. The flash takes about 30 seconds.

03

Wait for the reboot

The board restarts automatically when flashing is complete. You'll see SYSTEM BOOT on the display.

INSTALL SUGARFOOT'S RADAR Your browser doesn't support Web Serial. Please use Chrome.

Installs firmware v4 · ESP32 only · Download .bin instead

After installing

  • 01 On your phone, connect to the WiFi network called Sugarfoots Radar Setup
  • 02 Open a browser and go to 192.168.4.1
  • 03 Enter the PIN 111, select your WiFi network, enter your coordinates and range, then hit Save & Reboot
  • 04 Within 5 seconds of rebooting, planes start appearing on the radar
  • 05 D22 button — short press cycles declutter levels · 2 second hold returns to setup
